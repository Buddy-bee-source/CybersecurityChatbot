# CyberBot - Cybersecurity Awareness Chatbot

A WPF-based cybersecurity awareness chatbot that educates users about online safety topics through an interactive GUI.

## Features Implemented in Part 2
- WPF GUI with header, chat display, input box and send button
- ASCII art displayed in header on startup
- Voice greeting on startup
- Keyword recognition for 11 cybersecurity topics
- Random responses per keyword
- Sentiment detection (Worried, Curious, Frustrated, Happy)
- Memory and Recall -remembers name and favourite topics
